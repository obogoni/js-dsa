# Data Structures and Algorithms in JavaScript

A collection of simple implementations of basic data structures and algorithms in JavaScript.

## How to run the tests

1. Install the dependencies

```bash
npm install
```

2. Run the tests

```bash
npm test
```

## Data Structures

- Doubly linked list
- Linked list
- Queue
- Stack

## Algorithms

- Divide and Conquer
- Two Pointers
- Binary Search

## Problems

- Palindrome Check
- Crystal Balls
